Acerca de

Esta aplicación de búsqueda de películas forma parte de mi portafolio personal y ha sido desarrollada como parte de mi formación en Angular 19. El objetivo principal de este proyecto es consolidar mis conocimientos en el framework, aplicando conceptos clave como la creación de componentes reutilizables, manejo de servicios, enrutamiento y consumo de APIs externas.

La aplicación permite buscar películas por título y visualizar información detallada de cada una de ellas. La información es obtenida de la API de The Movie Database.
